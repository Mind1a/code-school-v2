## ğŸ§± Corresponding Mongoose Schema (Homework)

```js
import mongoose from "mongoose";

const homeworkSchema = new mongoose.Schema(
  {
    courseId: {
      type: String,
      required: true, // e.g. "python" or "html"
    },
    chapterId: {
      type: String,
      required: true, // e.g. "2.2"
    },
    homeworkId: {
      type: Number,
      required: true,
    },
    homeworkTitle: {
      type: String,
      required: true,
    },
    description: {
      type: String,
    },
    help: {
      type: String,
    },
    exampleCode: {
      type: String,
    },
    answer: {
      type: String,
    },
    hints: [
      {
        type: String,
      },
    ],
    resources: [
      {
        title: String,
        url: String,
      },
    ],
  },
  { timestamps: true }
);

export default mongoose.model("Homework", homeworkSchema);
```

## ğŸ§­ Backend Route Map

| Method     | Endpoint                                                       | Description                     |
| ---------- | -------------------------------------------------------------- | ------------------------------- |
| **GET**    | `/courses/:courseId/chapters/:chapterId/homeworks`             | Get all homeworks for a chapter |
| **GET**    | `/courses/:courseId/chapters/:chapterId/homeworks/:homeworkId` | Get one homework                |
| **POST**   | `/courses/:courseId/chapters/:chapterId/homeworks`             | Create a new homework           |
| **PATCH**  | `/courses/:courseId/chapters/:chapterId/homeworks/:homeworkId` | Update a homework               |
| **DELETE** | `/courses/:courseId/chapters/:chapterId/homeworks/:homeworkId` | Delete a homework               |
