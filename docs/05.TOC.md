## ­ЪћЌ Structure Summary

| Field              | Description                                |
| ------------------ | ------------------------------------------ |
| `courseName`       | Course name                                |
| `courseCode`       | Used in route like `/courses/:courseCode/` |
| `totalChapters`    | Number of chapters in this course          |
| `tableOfContent[]` | List of all chapters                       |
| `chapterNumber`    | Logical chapter index (e.g. 2.2)           |
| `title`            | Title of the chapter                       |
| `shortDescription` | Short preview text                         |
| `homeworks`        | How many homeworks chapter has             |
